# Sofia's Endless Runner

A browser-based endless runner game built with Phaser 3.

## Play

Once GitHub Pages is enabled, play at:  
`https://vvostryakov.github.io/sofia-s-endless-runner/`

## Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Lane up | `↑` or `W` | Swipe up |
| Lane down | `↓` or `S` | Swipe down |

## Enable GitHub Pages

1. Go to **Settings → Pages** in this repo
2. Set **Source** to `GitHub Actions`
3. Push any commit — the workflow deploys automatically

## Dev

No build step. Edit `game.js` and push — the game updates live.
